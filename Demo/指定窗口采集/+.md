在已有的“指定窗口采集”代码基础上，修改为只采集窗口的客户区（Client Area），去除标题栏和边框。

要求：

技术：

* 使用 win32gui.GetClientRect 获取客户区大小
* 使用 win32gui.ClientToScreen 将客户区坐标转换为屏幕坐标
* 使用 mss 只截取客户区区域

实现逻辑：

1. 获取窗口句柄 hwnd
2. 获取客户区尺寸 (width, height)
3. 获取客户区左上角在屏幕中的坐标
4. 构造 mss 的 monitor：
   {
   "left": x,
   "top": y,
   "width": width,
   "height": height
   }
5. 每一帧都重新获取坐标（防止窗口移动）

功能要求：

* 实时显示截图（cv2.imshow）
* 按 q 退出
* 保证画面没有窗口边框或标题栏

代码要求：

* 完整可运行
* 单文件
* 添加必要注释说明“窗口坐标 vs 客户区坐标”

可选调试：

* 在控制台打印客户区坐标
