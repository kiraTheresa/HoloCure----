目标：实现一个Python Demo，能够根据窗口标题，持续采集该窗口区域画面，并用OpenCV实时显示。

技术要求：

- 使用 Python
- 使用 mss 进行截图
- 使用 pywin32（win32gui）获取窗口信息
- 使用 OpenCV（cv2）显示画面
- 使用 numpy 处理图像

功能需求：

1. 根据窗口标题查找窗口句柄（hwnd）
2. 获取窗口位置（left, top, right, bottom）
3. 构造 mss 的 monitor 区域
4. 循环截图该区域
5. 转换为 numpy array
6. 使用 cv2.imshow 实时显示
7. 按下 'q' 键退出程序

注意事项：

- 如果窗口不存在，给出提示并退出
- 确保截图区域随窗口移动实时更新（每一帧重新获取窗口位置）
- 捕获的是窗口整体区域（暂不区分 client area）

代码要求：

- 写成一个完整可运行的 .py 文件
- 结构清晰，包含 main 函数
- 不需要类封装，简单函数结构即可
- 添加必要注释，说明每一步作用

额外优化（可选）：

- 在窗口上显示FPS
- 在控制台打印窗口坐标（调试用）

窗口标题示例：
"HoloCure"
